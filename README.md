# 🚀 30 Days of Machine Learning Challenge

Welcome to my **30 Days of Machine Learning** journey! 🎯 Over the past month, I've worked consistently to build 30 real-world, impactful, and end-to-end machine learning projects—1 project each day.

This initiative not only helped me sharpen my skills in supervised/unsupervised learning, computer vision, and NLP, but also challenged me to stay consistent and productive.

Each project includes:
- 📊 Data preprocessing & EDA
- 🧠 Model training & evaluation
- 📈 Performance metrics
- 📁 Clean and structured GitHub repository

---

## 🧰 Tech Stack & Tools

- **Languages:** Python
- **Libraries:** scikit-learn, pandas, NumPy, TensorFlow/Keras, OpenCV, XGBoost, Matplotlib, Seaborn
- **Visualization:** Matplotlib, Plotly, Seaborn
- **Deployment:** Streamlit, Flask *(in some projects)*
- **Notebooks:** Jupyter Notebook / Google Colab
